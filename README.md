# Dzzax Discord Bot

بوت ديسكورد قوي بعدة مميزات:
- أوامر Slash
- ترحيب بالصور
- تفاعل مع الصور والفيديوهات
- كشف الروابط وعقاب المخالفين (Timeout)
- تغيير الحالة بشكل تلقائي
- والمزيد...

## طريقة التشغيل
1. ثبت المكتبات:
```bash
npm install
```
2. أنشئ ملف `.env` وضع فيه التوكن الخاص بك.

3. شغل البوت:
```bash
npm start
```
